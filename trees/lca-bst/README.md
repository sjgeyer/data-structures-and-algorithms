# Lowest Common Ancestor in Binary Search Tree

## Challenge
Write a function that takes in a binary search tree and two values and finds the lowest common ancestor of those values.

## Example

Input:

           tree                         values
           
           [10]                          1, 8
          /   \
        [7]   [16]
       /   \     \
     [1]   [8]   [20]
                 /
               [17]
     
Output:

    7
