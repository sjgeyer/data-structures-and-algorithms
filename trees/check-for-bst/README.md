# Check for Binary Search Tree

## Challenge

Write a function that takes in a binary tree and returns `true` if the tree is a binary search tree, or `false` if it is not.

## Examples
          
           [10]
          /   \
        [3]   [11]          // =>   true
       /   \     \
     [1]   [9]   [20]
     
         [16]
         /   \
      [13]    [1]          // =>   false
      /   \     \
    [1]   [9]   [20]
