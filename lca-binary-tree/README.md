# Lowest Common Ancestor in Binary Tree

## Challenge
Write a function that takes in a binary tree and two values and finds the lowest common ancestor of those values.

## Example

Input:

           tree                         values
           
           [3]                           5, 4
          /   \
        [3]   [16]
       /   \     \
     [4]   [5]   [20]
     
Output:

    3
